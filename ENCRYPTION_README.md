# نظام تشفير معرفات المنتجات

## نظرة عامة
تم تطبيق نظام تشفير متقدم لحماية معرفات المنتجات في الموقع، مما يوفر طبقة إضافية من الأمان ويمنع الوصول غير المصرح به للمنتجات.

## الميزات
- **تشفير قوي**: استخدام خوارزمية AES-256-CBC للتشفير
- **أمان عالي**: مفتاح تشفير سري محمي
- **توافق مع النظام القديم**: يدعم الروابط القديمة للتوافق العكسي
- **سهولة الاستخدام**: دوال بسيطة للتشفير وفك التشفير

## الملفات المضافة/المعدلة

### ملفات جديدة:
- `inc/encryption.php` - ملف التشفير الرئيسي

### ملفات معدلة:
- `buy-now.php` - تحديث لاستخدام النظام المشفر
- `index.php` - تحديث الروابط لاستخدام المعرفات المشفرة
- `product-category.php` - تحديث الروابط لاستخدام المعرفات المشفرة
- `incomplete-orders.php` - تحديث الروابط لاستخدام المعرفات المشفرة
- `search-result.php` - تحديث الروابط لاستخدام المعرفات المشفرة

## كيفية الاستخدام

### تشفير معرف منتج:
```php
$encrypted_id = encrypt_product_id(125);
```

### فك تشفير معرف منتج:
```php
$product_id = decrypt_product_id($encrypted_id);
```

### إنشاء رابط آمن:
```php
$secure_link = create_secure_product_link(125);
// النتيجة: buy-now.php?id=ZCYNYYci6xq5qGiDfHexeFhUWXVTWkI4akVRMU1hWVluTTIxRHc9PQ%3D%3D
```

### التحقق من صحة المعرف المشفر:
```php
$is_valid = is_valid_encrypted_id($encrypted_id);
```

## مثال على التحويل

### الرابط القديم:
```
http://localhost/ecommerce/buy-now.php?id=125
```

### الرابط الجديد المشفر:
```
http://localhost/ecommerce/buy-now.php?id=ZCYNYYci6xq5qGiDfHexeFhUWXVTWkI4akVRMU1hWVluTTIxRHc9PQ%3D%3D
```

## الأمان
- **مفتاح التشفير**: محفوظ في ملف التشفير ويمكن تغييره حسب الحاجة
- **خوارزمية التشفير**: AES-256-CBC مع متجه تهيئة عشوائي لكل عملية تشفير
- **التحقق من الصحة**: فحص شامل لصحة البيانات المشفرة قبل فك التشفير

## التوافق العكسي
النظام يدعم الروابط القديمة، فإذا فشل فك التشفير، سيحاول استخدام المعرف مباشرة للتوافق مع الروابط الموجودة.

## الصيانة
- يمكن تغيير مفتاح التشفير في ملف `inc/encryption.php`
- جميع الروابط الجديدة ستستخدم النظام المشفر تلقائياً
- الروابط القديمة ستعمل بشكل طبيعي

## ملاحظات مهمة
- تأكد من نسخ احتياطي للموقع قبل التطبيق
- اختبر النظام في بيئة التطوير أولاً
- راقب ملفات السجل للتأكد من عدم وجود أخطاء
